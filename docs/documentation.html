‚öôÔ∏è Build Tools

Technol is build using different task to automate the development process. So for Getting started, you need to follow the below steps.

Note: This step is optional. This step is only for the advanced user who wants to speed up their development process with professional tools like gulp, nodeJS, etc. If you are not familiar with these tools and have no time to dive in, you can still use plain HTML / CSS / JS to customize a template. Files you need to edit are located inside the template folder. In this case, you can skip this and the next sections.

Installing Node Js

To use our build tools, you will need to download and install Node.js. If you do not have Node.js installed in your OS, you can download the package installer from the official website. Please download the stable version of Node.js (LTS).

Download Node.js
Installing Gulp.js

After installing Node in your system, you can proceed to Gulp installation. Run the below command in your terminal, which will install Gulp globally.

npm install gulp-cli -g

If you are using macOS, please use sudo keyword in the command because they need administrator rights to install Gulp globally.

sudo npm install -g npm
Installing NPM modules

First, change the command line path into your project where template folder is located. If you are newbie or you haven't work with the command prompt (CMD) before, you may check this article.

Once the path of your workflow changed to template folder, you may run the package.json file by using the following command:

npm install

If you check the project folder when the command has finished executing, you should see that Gulp has created a node_modules folder.

Run Gulp

Now you're ready to start building new pages. All you have to do is run gulp in Terminal by using the following command:

gulp

This will track all the sass file changes and start a local webserver at http://localhost:3000 While Gulp is running, Files in the HTML pages and scss files are monitored for changes, which will inject updated CSS or cause a refresh in Browsersync.

üîß Customization

Create your own styles without changing the template's core files so that you can take full advantage of future updates.

SCSS Method - _user.scss (recommended)

The sleekest way to make edits and additions to the template CSS is to use the provided _user-variables.scss and _user.scss files located in the scss folder.

In template/scss/ directory you will find two files dedicated for edit and additions:

  • Simply, copy the variables you need to edit from the main ..template/scss/_variables.scss and paste it in ..template/scss/_user-variables.scss It will automatically override the default values with the new ones.
  • To add your own CSS code, declare it in the _user.scss file.
Compiling changes

In order to see the changes, running Gulp command is required.

CSS Method - custom.css

If you prefer to work with the template as a static source without the aid of Gulp and Sass, you will need to add custom CSS through a separate CSS file.

We have provided a blank custom.css file in the template/css folder. To make your custom CSS active, you will need to add the following line after style.css in the <head> of each page:

<link rel="stylesheet" href="css/custom.css" />

üîé Favicon icon

Favicon is an icon associated with the URL that is displayed at various places, such as in a browser’s address bar or next to the site name in a bookmark list. There are many online favicon icon generators to generate .ico file. You can also use .png file.

You can add a Favicon to your Website using the following code, which already included in all HTML files head tag:

<link rel="shortcut icon" href="assets/images/favicon.ico" />

The Logo can be found in the Header header tag. Replace "logo.svg" with your own logo image.

<!-- logo -->
<a class="navbar-brand d-lg-block text-lg-center" href="index.html">
  <img src="images/logo.svg" alt="Logo">

How to change logo height?

You can set the height according to your logo type and your requirement.

CSS

To set logo height open css/style.css file and change height value in .navbar-brand img class, default value of logo height is 40px.

SCSS

To set logo height open scss/custom/navbar.scss file and change height value in .navbar-brand img class, default value of logo height is 40px.

If you change logo height then header height will automatically increase and the menu will set vertically center.

üé® Color Schemes

Follow super easy options to change your website color.

Option 1: Customize global colors via Sass variables

If you want to create your color scheme - the fastest and easiest way to do that is via Sass variables.

Simply copy necessary variable from template/scss/variables.scss file and paste to template/scss/user-variables.scss, then compile it using gulp.

See example code part of Sass variables file responsible for global colors:

$primary:       #b2f9ea;

$theme-colors: ();
$theme-colors: map-merge(
  (
    "primary":    $primary,
    "secondary":  $secondary,
    "success":    $success,
    "info":       $info,
    "warning":    $warning,
    "danger":     $danger,
    "light":      $light,
    "dark":       $dark
  ),
  $theme-colors
);

$link-color:                theme-color("primary");
$link-hover-color:          darken($link-color, 15%);
$dropdown-link-active-bg:   theme-color("primary");
$pagination-color:          theme-color("primary");
$pagination-active-bg:      theme-color("primary");
$pagination-active-border-color: theme-color("primary");
Option 2: Replace global colors in CSS

We don't recommend this method, but if you prefer vanilla CSS, you can replace all color hex values inside template/css/style.css with your own using "find and replace" panel of your code editor.
